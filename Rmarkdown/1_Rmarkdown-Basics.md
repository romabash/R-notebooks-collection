### Plot *iris* data using *ggplot2*

  - Plotting Petal Length vs Petal Width, using Species for color

<!-- end list -->

``` r
library(ggplot2)

ggplot(data = iris) + 
geom_point(aes(x = Petal.Length, y = Petal.Width, color = Species))
```
